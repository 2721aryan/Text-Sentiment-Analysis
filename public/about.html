About This Project

Project Overview

This Text Sentiment Analysis web application analyzes the emotional tone of text input. It categorizes text as positive, negative, or neutral based on its content using natural language processing.

How It Works

The application uses TextBlob, a Python library for processing textual data, to analyze the sentiment of user-provided text. It calculates a polarity score between -1 (very negative) and 1 (very positive), with 0 being neutral.

Project Files

api/index.py

The main API entry point that handles HTTP requests, processes form submissions, and performs sentiment analysis using TextBlob.

public/index.html

The HTML page that provides the structure for the web interface, including the form for user input and the display of analysis results.

public/css/style.css

The CSS file that defines the dark-themed styling and layout of the application, including responsive design for different screen sizes.

vercel.json

Configuration file for Vercel deployment, specifying routing and build settings.

Technologies Used

This application uses Python for the backend sentiment analysis, HTML/CSS for the frontend, and TextBlob for natural language processing.
